第三章：HTML框架布局


	< table>表格布局。
	< div>层布局。
	< frameset>框架布局。（不在使用< body>标记。）
	【提供属性：cols左右分割。
	         border框架边框大小。
	         rows上下分割。
	         （例：< frameset cols="100，*，*">：
	          表示将窗口分为左中右三个窗口，其中左窗口宽为100像素，然后右面两个窗口平均分配！）
	         scrolling="no"不显示滚动条！
	         noresize="noresize"不可调整框架大小。】
	         target目标窗口属性：
	         （< a href=url target="_blank">显示在一个新打开的窗口中！）
	         （< a href=url target="_self">显示在相同的框架或窗口中打开目标页！）
	         （< a href=url target="_parent">显示在父窗口或者包含了超链接引用的框架的框架集。） 
	         （< a href=url target="_top">显示在当前浏览器窗口，并取消掉当前窗口框架。）
