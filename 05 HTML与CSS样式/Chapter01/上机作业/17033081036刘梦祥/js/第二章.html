第二章：HTML表格


	表格的基本结构：
	1.每一对< table>和< /table>表示一个表格。
	2.每一对< tr>和< /tr>表示单独一行。
	3.每一对< td>和< /td>表示一个单元格。
	【提供border属性来设置边框线。】
	单元格的合并：
	colspan属性：横向合并。
	rowspan属性：纵向合并。
	表格的对齐：
	【提供align属性。】
	表格的填充属性与间距属性：
	cellpadding：填充属性，代表每个单元格内部的内容与外面的边框之间的距离。
	border（边框的厚度。）
	cellspacing（单元格间距。）
	表格的修饰与美化：
	bgcolor属性为表格设置背景色。
	backround属性为表格设置背景图片。
