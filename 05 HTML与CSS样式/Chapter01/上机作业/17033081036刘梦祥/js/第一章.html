第一章：基本标记


< html>----------标记文档的开始
< head>----------标记文档的头部
< title>构成标题< /title>
< /head>---------标记文档的尾部
< boby>
（主要内容）
< /boby>
< /html>---------标记文档的结束！
	

超链接语法：< a href="链接到的地址">链接内容（可以是文字也可以是图片）< /a>
标题标签：< hn>内容< /hn>
段落标签：< p>内容< /p>
【提供align属性：left（左对齐）
	        center（居中）
	        right（右对齐）
	        默认值为left{如：内容< /h2>}】
预排版标签：< pre>内容< /pre>(原样显示)
字体标签：< font>内容< /font>
【提供属性：face：代表字体名称；如："宋体"，"黑体"等字体！
	   size：设置字体大小 1<=size<=7
	   color:设置字体颜色!
	   如:< font color="blue" size=6 face=微软雅黑>内容< /font>】
字体风格标签：< b>加粗< i>倾斜< u>下划线< s>删除线
	     < strong>强调 < small>较小< big>较大< sup>上标< sub>下标
水平线标签：< hr>【提供属性：align、size、width、color】
滚动字幕标签：< marpuee>内容< /marquee>
【提供属性：direction:设置滚动方向（left：表示向左；right：表示向右；
up：表示向上；down：表示向下！）
	   width：设置滚动字幕的宽度。
	   height：设置滚动字幕的高度。
	   bgcolor：设置滚动字幕的背景颜色。
	   hspace和vspace：分别用于设定滚动字幕的左右边框和上下边框的宽度。
	   scrollamount：用于设定活动字幕两次之间的滚动距离即滚动速度，
       值越大，越快！
	   scrolldelay：用于设置两次滚动间的延迟时间。】
图像标签：< img src="图片地址"alt="提示文字"title="提示文字"/>
【提供属性：src：设置图片所在路径。
	   width与height：设置图片的宽度和高度。
	   border：图片周围出现的边框。
	   align：设置图片相对于文字的对齐方式。
	   （可取值：top,middle,bottom,left,right,默认为：bottom）
	   bgcolor：设置滚动字幕的背景颜色。】
有序列表：< ol>内容< /ol>
【提供属性：type（可取值："A"、"a"、"I"、"i"、"1"）】
无序列表：< ul>内容< /ul>
【提供属性：type（可取值："square(方形)"、"disc（圆圈）"、"circle（点）"】
