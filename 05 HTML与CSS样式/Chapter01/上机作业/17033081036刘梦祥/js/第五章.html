第五章：表单与表单控件


< form>表单标签
	【提供属性：name代表表单名称
	           method代表表单提交的方法
	          （可取值：get/post）
	           action代表表单数据进行处理的目标地址】
< input/>表单控件标签
        【提供属性：type表单控件
                 （可取值：text单行文本输入
                          password密码框
                          radio单选按钮
                          checkbox复选框
                          select下拉列表
                          reset重置按钮
                          submit提交按钮）
                          textarea多行文本框】
 文本框控件text
        【提供属性：name代表文本框控件的名字
                   size代表文本框的宽度大小
                   maxlength代表最多容纳的字符数
                   readonly代表此控件为只读】
 密码框控件password
         【提供属性：name代表文本框控件的名字
                    value代表文本框的默认值
                    size代表文本框的宽度大小
                    maxlength代表最多容纳的字符数
                    readonly代表此控件为只读】
 单选按钮框控件radio
         【提供属性：name单选按钮控件的名称
                    value单选按钮的初始值
                    checked代表此单选按钮是否被选中
                    （可取值：true代表选中
                             false代表没被选中）】
 复选框控件：checkbox
         【提供属性：name复选按钮控件的名称
                    value复选按钮的初始值
                    checked代表此复选按钮是否被选中
                    （可取值：true代表选中
                             false代表没被选中）】
 普通按钮控件：button
         【提供属性：name代表按钮的名字
                    value代表按钮上的文本】
