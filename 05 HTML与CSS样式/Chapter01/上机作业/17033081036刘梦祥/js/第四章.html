第四章：浮动窗口与多窗口


			< iframe>浮动标签
			【提供属性：src代表此浮动窗口所引用的页面，一般采用相对路径指定。
			         width/height代表此浮动窗口的宽度和高度。
			         name代表此框架的名称。
			         scolling代表此浮动窗口是否会出现滚动条。
			             （可取值：auto代表会根据浮动窗口中的内容多少取决是否出现滚动条。
			                      yes代表无论出现多少内容都会出现滚动条。
			                      no代表无论出现多少内容都不会出现滚动条。 ）
			         frameborder代表此浮动窗口是否会出现边框。
			             （可取值：1代表会出现边框。
			                      0代表不会出现边框。）】
			< fieldset>分组标签。
				      （< legent>是为了在分组标签中放入标题。）
			< bgsound>音乐标签
			【提供属性：src表示要播放的背景音乐的文件路径。
			         loop循环播放的次数：
     			              (可取值：-1表示一直不断的播放
			                      0表示只播放一次)
			                      也可以取任意值代表播放的次数】
			< embed>多媒体标签
			【提供属性：src表示要播放的背景音乐的文件路径。
			         autostart设置是否在多媒体下载完成后就自动播放，
			             （可取值：true下载完成后自动播放
			                      false下载完成后不自动播放--默认值）
			         hidden是否完全隐藏控制面板
		                     （可取值：true可以隐藏
		                              false不能隐藏）】
			< img>标签
			【提供属性：dynsrc设置多媒体文件的路径名。
			         loop设置多媒体播放的循环次数。
			         start设置文件的播放方式。
                 	              （可取值：fileopen表示网页打开时即播放，
			                       mouseover表示当鼠标滑到多媒体上时播放。）】
